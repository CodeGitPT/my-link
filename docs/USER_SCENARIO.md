# 사용자 시나리오 (User Scenarios) - 마이링크

마이링크 서비스 내에서 발생할 수 있는 모든 사용자(방문자 및 소유자)의 행동 패턴을 정의합니다.

## 1. 방문자 시나리오

1.  **프로필 접속:** 방문자는 특정 사용자의 온라인 활동을 한곳에서 일괄 확인하기 위해 외부 서비스(인스타그램, 유튜브 등)를 통해 공유받은 마이링크 주소(`mylink.com/displayname`) 링크를 클릭하거나 직접 입력하여 접속한다.
2.  **프로필 인지:** 방문자는 화면 최상단에 렌더링된 사용자 표시 이름(Username)과 간단한 소개글(Bio)을 읽고 자신이 의도한 올바른 페이지에 도착했음을 확인한다.
3.  **링크 탐색:** 방문자는 원하는 채널로 이동하기 위해 메인 화면에 나열된 버튼들을 탐색하며, 각 버튼 좌측의 파비콘(아이콘)과 텍스트 제목(Title)을 통해 도착지가 어디일지 유추한다.
4.  **링크 클릭 및 카운트:** 방문자는 원하는 목적지로 이동하기 위해 목적지 버튼을 클릭하며, 시스템은 향후 소유자에게 통계 피드백을 제공하기 위해 **이 링크 클릭(Click) 이벤트를 카운트하여 데이터베이스에 기록**한다.

## 2. 소유자 시나리오

### 2.1 계정 관리 및 온보딩
1.  **회원가입/로그인:** 소유자는 마이링크 서비스를 이용거나 관리하기 위해 메인 페이지에서 구글 소셜 로그인 버튼을 클릭하여 즉각적으로 온보딩/인증한다.
2.  **URL 슬러그 자동 생성:** 소유자가 최초 가입 시, 시스템은 구글 이메일 주소의 앞부분(ID)을 추출하여 고유 주소 슬러그(DisplayName)로 자동 할당하여 페이지를 생성해 줌으로써 과정의 피로를 덜어준다.
3.  **로그아웃:** 소유자는 공용 기기 등에서 본인 계정을 보호하기 위해 화면 내 로그아웃 버튼을 눌러 안전하게 세션을 종료한다.
4.  **회원 탈퇴:** 소유자는 더 이상 서비스를 운영하지 않기 위해 설정 메뉴에서 계정 탈퇴를 신청하며, DB 내 본인의 모든 링크와 프로필 데이터를 영구적으로 지운다.

### 2.2 인증 및 공유
1.  **편집 모드 진입:** 소유자는 로그인 성공 후 본인의 마이링크 프로필 화면으로 이동되며, 페이지 내 프로필/링크 텍스트들이 인라인 수정이 가능한 형태로 변환된 것을 보고 자신이 페이지 관리 상태임을 인지한다.
2.  **내 링크 복사:** 소유자는 생성된 마이링크 주소를 외부 채널 바이오(Bio)에 기입하기 위해 화면 상단에 제공된 [내 링크 복사] 버튼을 클릭하여 주소를 클립보드에 빠르고 쉽게 담는다.

### 2.3 프로필 관리 (인라인 편집)
1.  **표시 이름(Username) 수정:** 소유자는 방문자에게 노출되는 본명을 더 친근하게 변경하기 위해 이름 텍스트 영역을 직접 클릭해 인라인 편집 상태를 활성화한 뒤, 새 이름을 입력하고 엔터(Enter) 키나 여백을 클릭하여 데이터베이스에 즉시 저장한다.
2.  **소개글(Bio) 수정:** 소유자는 자신을 어필할 소개 문구를 업데이트하기 위해 소개글 항목을 직접 클릭하여 자유롭게 수정한 후, 엔터(Enter) 키로 마찬가지로 실시간 반영/저장한다.

### 2.4 링크 데이터베이스 관리
1.  **새 링크 추가:** 소유자는 자신의 새로운 채널이나 홍보물을 리스트에 띄우기 위해 하단 또는 상단의 [새 링크 추가] 빈 필드에 목적지 주소(URL)와 노출할 제목(Title)을 입력하고 완료한다.
2.  **파비콘 연동 확인:** 소유자는 자신이 등록한 사이트의 성격이 버튼에 잘 어필되는지 파악하기 위해, 주소가 입력되자마자 구글 API에 의해 대상 사이트 아이콘이 버튼 좌측에 삽입·동기화되는 것을 시각적으로 감상한다.
3.  **링크 인라인 수정:** 소유자는 과거에 등록했던 특정 링크의 제목이나 도착지 URL에 오타가 있음을 발견하고, 해당 링크 블록 안의 텍스트를 직접 클릭해 텍스트를 수정 후 엔터(Enter)로 확정 짓는다.
4.  **링크 삭제:** 소유자는 현재 접속 불가하거나 프로모션이 끝난 링크를 화면에서 내리기 위해, 해당 링크 우측에 위치한 [삭제 아이콘(또는 버튼)]을 눌러 되돌릴 수 없음을 컨펌한 뒤 리스트에서 완전히 제거한다.
